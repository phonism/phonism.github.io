常用快捷键总结

2016-07-19 • 标签: Linux

主要记录一些我平时常用的快捷键

Linux命令行快捷键

  • Ctrl + a: 光标移动到当前行的行首
  • Ctrl + w: 光标移动到当前行的行末
  • Ctrl + p: 显示上一条命令
  • Ctrl + n: 显示下一条命令
  • Ctrl + f: 光标向前移动一个位置
  • Ctrl + b: 光标向后移动一个位置 (因为我Ctrl + b 绑定了tmux的快捷键，所以这里需要按两下Ctrl + b)
  • Ctrl + r: 然后输入若干字符，开始向上搜索包含该字符的命令，继续按Ctrl+r，搜索上一条匹配的命令
  • Ctrl + d: 删除光标所在处的字符
  • !?str?: 执行包含str的最新的命令

Vim常用快捷键

  • 区块操作（主要用于批量注释删除注释等）: 在命令模式下Ctrl + v然后用hjkl选择区域，按I进入插入模式，然后进行操作，操作完成后按Esc。
